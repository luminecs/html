在 HTML 中，<a> 标签用于创建超链接，即指向互联网上其他网页、文件或资源的链接。<a> 中的“a”代表“锚点”。

Source Code

<a href="https://luminecs.com">Visit luminecs.com!</a>

Output

Visit luminecs.com!

在上面的例子中，“http://luminecs.com;”是链接指向的 URL 或网址。“Visit luminecs.com!”文本是用户将在网页上看到的内容，当他们点击它时，他们将被带到 luminecs.com 网站。

该<a>标签还可用于创建指向网页特定部分的链接，称为锚链接。要创建锚链接，您需要为要链接的元素指定“id”属性，然后使用该 id 作为标签中的 href 值<a>。

以下是如何创建锚链接的示例：

Source Code

<h2 id="section1">第 1 节</h2>
<p><a href="#section1">转至第 1 部分</a></p>

Output

第 1 节

转至第 1 部分

在上面的示例中，<h2>标签的“id”属性设置为“section1”，这会在页面上的该点处创建一个锚点。标签<a>的 href 值为“#section1”，链接到该锚点。当用户点击该链接时，页面将滚动到 ID 为“section1”的部分。

该<a>标签还可以与其他 HTML 元素（例如图像或按钮）结合使用，以创建可点击的链接。
